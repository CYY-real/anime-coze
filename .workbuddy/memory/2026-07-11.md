## 第五轮: 平台 logo 升级 (网络搜 SVG 资源)

### 用户反馈
"几个平台的 logo 你网络上搜搜看有没有 svg 资源呗"

### 搜索结果
- **bilibili**: simple-icons (24×24 mono) - https://dl.svgcdn.com/svg/simple-icons/bilibili.svg ✓
- **iqiyi**: arcticons (48×48 line) - https://dl.svgcdn.com/svg/arcticons/iqiyi.svg ✓
- **migu**: arcticons (48×48 line) - https://dl.svgcdn.com/svg/arcticons/migu.svg ✓
- **youku**: brandico (1000×1000 mono) - https://dl.svgcdn.com/svg/brandico/youku.svg ✓
- **tencent video**: 公开 CDN 都被 Cloudflare/svgrepo 挡了, 官方下载页 iqiyi.com 给 PNG/PDF 不给 raw SVG. **自己画** (48×48) - 三色播放三角 橙 #FA8036 + 绿 #74CA12 + 蓝 #168CF5, 中间白色播放图标

### 改动
- `PLATFORM_BRAND`: 全部 SVG 替换为网络 raw 资源
- **关键: 所有 SVG 改成内嵌具体颜色** (替换原 currentColor) — 这样可以去掉外层 background 染色
- 5 处 `style="background:${color}"` 全部去除 (renderMylist platform-tag / openPlatformPopup card+list / editPlatforms row)

### 关键设计取舍
- **不用图片 URL 引用, 全部内联 SVG**: 1) 离线可用, 2) 不受 CDN 失效影响, 3) 可直接 CSS 控制 fill/stroke
- **不用 background 染色, 颜色内嵌进 SVG**: 适配 mono SVG (youku) 和 line SVG (iqiyi/migu) - 它们只有 stroke/fill=currentColor, 必须 inline 改具体色
- **爱奇艺短字符 '奇' → '爱'**: 之前我用色块字符 '奇' 误导了用户, 改为 '爱' (与官方品牌一致)
- **bilibili 品牌色 'FB7299' → '00A1D6'**: 之前我用了粉红 (小程序的 banner 渐变色), 但 B 站品牌主色是蓝色 #00A1D6. 这是 simple-icons 用的颜色, 更准

## 第三轮: 顶部标题"飘"问题 (用户拍板: 不置顶)

### 用户反馈
"向下滑动的时候, 追番不迷路的标题一直在顶部, 但是又没有背景, 要么不要置顶, 要么加个背景"

### 诊断 (避免下次犯同样错)
- **第一遍误诊**: 我以为是 `.mylist-header` (二级标题"我的追番")
- **实际情况**: 是顶部的 `.header` (一级标题"✨ 追番不迷路")
- `.header` 原本是 `position:sticky;top:0;z-index:10;background:transparent;` —— 真的粘在顶部, 背景透明, 所以"飘"
- 教训: 看到"顶部飘着的元素", 第一反应应该是全局 grep `position:sticky` 找源头, 而不是凭 CSS 类名猜

### 改动
- `.header`: 去掉 `position:sticky;top:0;z-index:10;`, 改为 `position:relative;z-index:auto;`
- 顶部标题现在跟内容一起滚走 (普通文档流)
- 同步确认全文已无 `position:sticky` 残留 (grep 验证 0 匹配)

### 用户最终选择
"不置顶, 正常随内容滚走" —— 简单方案, 不破坏现有布局
